# Ajav (ALPHA)

Ajav is friv alternative with C/Raylib

## Games

### WeakPompaelo
![WeakPompaelo](resources/games/weakpompaelo.png)

Based in PowerPoplona

## Build and compilation

**Dependecies:**

```
BUILD: raylib 5.5+ PLATFORM_WEB
LOCATION: MAIN_FOLDER

BUILD: emsdk 4.0.23
LOCATION: MAIN_FOLDER
```

Run compile.sh to compile the main menu

Run ../game_compile.sh in game folder to compile the game

**Optional:**

Change title website: raylib/src/minshell.html -> ```<title>```
Change css game: raylib/src/minshell.html -> ```<style> -> body```

## Run

Check [Raylib Step 6](https://github.com/raysan5/raylib/wiki/Working-for-Web-(HTML5)#6-test-raylib-game-on-web)
